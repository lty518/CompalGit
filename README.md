***
## Edge Game Server
* Python 環境及套件安裝
    安裝完Python 3之後，還要安裝相關套件
* 安裝Flask


## Stream Server
* Python 環境及套件安裝
    安裝完Python 3之後，還要安裝相關套件
* 安裝Flask

# 備註
Game Server 以及 Stream Server 可以放在同一台電腦上，由Game Server把遊戲畫面 推流到 Stream Server所host的 NginX Server
